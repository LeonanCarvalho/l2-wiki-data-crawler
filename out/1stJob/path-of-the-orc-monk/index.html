Path of the Orc Monk
Level: 19+
Quest type
Restrictions: None
Race: Orc
Class: Orc Fighter
Start location Orc Village / Schuttgart Territory
Start NPC Gantaki Zu Urutu
Reward 80,314
5,087


Description

  1. Talk to Gantaki Zu Urutu in the Orc Village.

    Map

  2. Talk to Khavatari Rosheek near the Frozen Waterfalls.

    Map

  3. Go to the Frozen Waterfalls, kill Kasha Bear with fist weapon only, until you have Kasha Bear Claw  — 5 pcs..

    Map

  4. Talk to Khavatari Rosheek near the Frozen Waterfalls.

    Map

  5. Go to the Frozen Waterfalls, hunt Kasha Blade Spider using fist weapon only, until you have Kasha Blade Spider Talon  — 5 pcs.

    Map

  6. Talk to Khavatari Rosheek near the Frozen Waterfalls.

    Map

  7. Go to the Cave of Trials, kill Scarlet Salamander using fist weapon only, you need to get Scarlet Salamander Scale  — 5 pcs.

    Map

  8. Talk to Khavatari Rosheek near the Frozen Waterfalls.

    Map

  9. Talk to Gantaki Zu Urutu in the Orc Village.

    Map

  10. Talk to Prefect Kasman in Orc Guild in Gludin Village (you can teleport to Gludin Village from Orc Village for free via Gatekeeper Tamil).

    Map

  11. Talk to Khavatari Toruku near Orc Barracks.

    Map

  12. Khavatari Toruku asks you to kill different warriors. You need to each by 3 mobs of each kind, to collect proof and to deliver it to Khavatari Toruku. You can use fist weapon only. Goals: Windmill Hill - Vuku Orc Warrior; Ruins of Agony or Ruins of Despair - Ratman Warrior; Langk Lizardmen Dwellings - Langk Lizardman Warrior; Fellmere Harvesting Grounds - Felim Lizardman Warrior. You need to get Vuku Orc Tusk  — 3 pcs., Ratman Fang  — 3 pcs., Langk Lizardman Tooth  — 3 pcs., Felim Lizardman Tooth  — 3 pcs.

    Map

  13. Talk to Khavatari Toruku near Orc Barracks.

    Map

  14. Talk to Prefect Kasman in Orc Guild in Gludin Village and get the reward.

    Map

  15. When you achieve level 20, talk to High Prefect Osborn in Orc Guild in Gludin Village or to High Prefect Drikus in Orc Guild in the Town of Gludio to transfer your class to Monk.

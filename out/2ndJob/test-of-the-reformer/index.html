Test of the Reformer
Level: 39+
Quest type
Restrictions: None
Race: Human, Dark Elf
Class: Cleric, Shillien Oracle
Start location Town of Giran / Giran Territory
Start NPC Pupina
Reward 151,200


Description

  1. Talk to Priestess Pupina in Einhasad Temple in the Town of Giran.

    Map

  2. Go to the Execution Grounds, hunt quest monsters Nameless Revenant (use the Disrupt Undead skill), until you have Ripped Diary  — 7 pcs.

    Map

  3. Quest Monster Aruraune will appear. Kill it and get Huge Nail .

    Map

  4. Talk to Priestess Pupina in Einhasad Temple in the Town of Giran.

    Map

  5. Talk to Preacher Sla on the Plains of Dion and agree to help her.

    Map

  6. Talk to Katari near Swampland. 2 quest monsters will appear: Ol Mahum Inspector and Ol Mahum Pilgrim, kill Ol Mahum Inspector.

    Map

  7. Talk to Katari once again.

    Map

  8. Quest monster Ol Mahum Betrayer will appear. Kill it (it will try to escape, use "Dryad Root" to hold it) and get Letter of Betrayer .

    Map

  9. Talk to Katari.

    Map

  10. Talk to Preacher Sla on the Plains of Dion.

    Map

  11. Talk to Kakan at the Immortal Plateau, Southern Region.

    Map

  12. Ol Mahum Pilgrim and Quest Monster Crimson Werewolf will appear. Wait until Ol Mahum Pilgrim is dead, then kill the werewolf using the Wind Strike skill.

    Map

  13. Talk to Kakan.

    Map

  14. Talk to Nyakuri near Abandoned Coal Mines.

    Map

  15. Ol Mahum Pilgrim and Quest monster Krudel Lizardman will appear. Kill Krudel Lizardman using the "Wind Strike" skill.

    Map

  16. Talk to Nyakuri.

    Map

  17. Talk to Ramus near the Singing Waterfall (not far from the Elven Ruins).

    Map

  18. Go to the Elven Ruins (you can get there via NPC Gatekeeper Siff, which is located near the blocked entrance to the ruins), hunt Silent Horror, Skeleton Marksman, Skeleton Lord, Misery Skeleton, Skeleton Archer (you can find the mobs in the northern parts of the ruins - the furthest part from the entrance), collect Bone Fragment  — 5 pcs. (one piece from each type of mobs).

    Map

  19. Talk to Ramus near the Singing Waterfall (not far from the Elven Ruins).

    Map

  20. Talk to Preacher Sla on the Plains of Dion and get the reward.

    Map
